# 运算符

## 算数运算符
bash不支持算数运算，但可以通过其他命令来实现，如`awk``expr`等
1. 加：`expr $a + $b`
2. 减：`expr $a - $b`
3. 乘：`expr $a \* $b`
4. 除：`expr $a / $b`
5. 余：`expr $a % $b`
> 乘法的*需要转义
> 运算符左右必须有空格

## 关系运算符
关系运算不支持字符串，只支持数字，除非字符串的值是数字，见[条件测试](条件测试.md)
