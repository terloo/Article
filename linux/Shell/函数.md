# 函数

## 语法
```sh
# 标准定义
function 函数名() {
    函数体
    return 返回值 # 可以省略
}

# 简略写法   使用function关键字时可以省略小括号
function 函数名 {
    函数体
    return 返回值 # 可以省略
}

# 精简写法  不能省略小括号
函数名 () {
    函数体
    return 返回值 # 可以省略
}

## 调用函数
函数名
```

## 局部变量
定义在函数中的变量，只能在当前函数中使用
```sh
函数名() {
    name='123'
}
```

## 参数
调用函数时，可以传入参数，在函数中使用`$n`来获取
```sh
函数名() {
    echo "传入的参数是，$1, $2, $3等等，一共有$#"
    echo "传入的参数是$*，一共有$#"
}
```
