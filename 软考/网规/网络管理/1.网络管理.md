# 网络管理

| 功能     | 说明                                                           |
| -------- | -------------------------------------------------------------- |
| 配置管理 | 负责监测和控制网络的配置状态                                   |
| 性能管理 | 网络通信信息的收集、加工和处理                                 |
| 故障管理 | 迅速发现、定位和排除网络故障，保证网络高可用性                 |
| 安全管理 | 保证网络管理系统正确运行，保护被管理的目标免受侵害和破坏       |
| 计费管理 | 正确计算和收取用户使用网络服务的费用，进行网络资源利用率的统计 |
