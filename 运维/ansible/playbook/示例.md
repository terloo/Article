# 示例

## 创建mysql用户
```yaml
- hosts: dbserver
  remote_user: root
  # gather_facts: no # 禁用信息收集
  tasks:
  - name: create group
    group: name=mysql system=yes gid=306
  - name: create user
    user: name=mysql shell=/sbin/nologin system=yes group=mysql uid=306 home=/data/mysql create_home=no
```

## 安装nginx
```yaml
- hosts: webserver
  remote_user: root
  tasks:
  - name: install nginx
    yum: name=nginx
  - name: enable nginx
    service: name=nginx state=started enabled=yes
```
