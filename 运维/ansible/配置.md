# 配置
ansible配置文件使用cfg格式

## default
default语句块保存了一些ansible的默认配置
| 配置项            | 说明                                   |
| ----------------- | -------------------------------------- |
| inventory         | 主机清单文件路径                       |
| remote_tmp        | 远程临时文件/命令目录                  |
| local_tmp         | 本机临时文件/命令目录                  |
| forks             | 并发数                                 |
| sudu_user         | sudo使用的用户                         |
| ask_sudo_pass     | 执行sudo命令时是否输入密码             |
| remote_port       | ssh端口                                |
| host_key_checking | 是否自动确认对远程服务器公钥的检查         |
| log_path          | 日志文件路径，默认/var/log/ansible.log |
| module_name       | 默认模块                               |
