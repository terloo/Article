# options
ansible使用到的命令选项

| 选项 | 解释                                        | 备注                    |
| ---- | ------------------------------------------- | ----------------------- |
| -i   | 指定主机清单文件                            |                         |
| -m   | 指定ansible模块                             |                         |
| -a   | 指定ansible动作，一般是模块中的选项或者命令 | 格式为"xxx option1=aa option2=bbb" |
